Download Py2D

Interested in Py2D? Great! Since Py2D is still under heavy development, there is no such thing as a stable version. However, you can download the current development version as a zip archive or tarball from GitHub.

Download .zip Download .tar.gz Py2D on GitHub

Git access

If you are interested in developing Py2D or want to always have access to the latest, bleeding-edge version, the git repository is the way to go for you: 

	$ git clone git://github.com/sseemayer/Py2D.git

I am looking forward to your pull requests!
